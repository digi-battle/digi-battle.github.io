This is a rotating can of Dr Pepper.

One day I was looking at a non-rotating can of Dr Pepper and thought to myself; "It sure would be great if this would rotate". Then I realised I needed a hobby, but decided instead to teach myself how to make an animated GIF.

The plan was to take pictures of the can rotating then remove the background using a green screen by chroma keying. This would create a stop-motion animation effect. Here is one of the original stills from this.

Unfortunately this caused an unforeseen issue where the green screen was visible in the reflections on the can, which didn't look good.

So I re-shot all the frames but with a white background.

The images were then imported into Sony Vegas as an image sequence.

Thankfully because the subject of the animation never changes shape, I could isolate it using a mask.

The last thing I did in Sony Vegas was render the video as an image sequence.

Which gave me a nice folder of alpha channelled PNGs of the Dr Pepper can.

All that was left to do was compile the images into an animated GIF. I did this using GIMP. The steps to do this are:

  • File > New
  • Enter the dimensions of one of the images
  • File > Open as Layers > Select your folder with the frames
  • Delete the Background layer
  • Image > Crop to Content
  • (At this point you can resize the image if you like)
  • File > Export As
  • Put .gif at the end of the file name and you'll see the GIF export window
  • Click Export

These are the GIF export settings that I used.

And that's it. Did I know what I was doing? Kind of. Is there a better way of achieveing this? Definitely. Do I now have a beautiful rotating Dr Pepper can to enjoy? Absolutely.
